# Cleanest Cleaners Landing Page

A simple, supportive landing page for Cleanest Cleaners - a cleaning company specialized in serving elderly clients.

## Features

- Clean, simple design with light blue, white, and light green color scheme
- Supportive and understanding tone throughout
- Sections for:
  - Hero/Introduction
  - About/Main Message (highlighting specialized care for elderly)
  - Services (placeholder for services list)
  - Contact Information (placeholder)

## Color Scheme

- **Primary Blue**: Light blue tones (#87CEEB, #4A90A4)
- **White**: Background and card backgrounds
- **Light Green**: Accent color (#90EE90) for feature highlights
